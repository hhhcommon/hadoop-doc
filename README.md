# 大数据相关文档

“hadoop-doc” 计划输出大数据相关的知识和理解，虽然叫“hadoop-doc”，但不只限于 hadoop 家族

计划主要涉及几个方面：

* 官方文档翻译
* 相关知识入门&总结性文档
* 大数据相关技术原理分析

这个 Repository 本人打算进行持续更新，希望最终能有一个系统性的知识输出
